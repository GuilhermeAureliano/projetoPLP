# Parte Lógica
Para rodar o projeto abra o terminal e rode os seguintes comandos abaixos:

`cd Desktop`

`git clone https://github.com/GuilhermeAureliano/projetosPLP.git`

`cd projetosPLP`

`cd Prolog`

`swipl -f .\main.pl`

OBS: Na hora do cadastro escrever tudo minúsculo para evitar bugs.
